<static> {string} JKIL.types.getClassName(pm_value, ph_options)
Gets the value of the hidden [[Class]] property for the given value, i.e. the “class” name.
       Syntax:
                JKIL.types.getClassName ( pm_value[, ph_options] )
       Example 1: 
                var myObject = new MyObject();
                var className = JKIL.types.getClassName ( myObject, {checkConstructor: false} ); // "Object"
       Example 2: 
                var myObject = new MyObject();
                var className = JKIL.types.getClassName ( myObject, {checkConstructor: true} ); // "MyObject"
Parameters:
{mixed} pm_value
The value whose "class" is to be determined.

{hash} ph_options Optional
A hash of options:

{boolean} ph_options.checkConstructor Optional, Default: false
If true, for objects the method checks whether pm_value is an instance of its constructor property (i.e. if constructor has not been altered) and returns the name of this constructor.
Returns:
{string} A string describing the [[Class]] property (for instance "Object", "Array", "String", "Boolean" and so on).
Notice: window returns "Window" even for Google Chrome.

<static> {boolean} JKIL.types.isEmptyString(pm_value, ph_options)
Checks whether the given value is an empty string literal or string object, that is a string having a 0 length.
If pm_value is a string literal or a string object, it can be trimmed on both sides before checking if the string length is 0 or not.
       Syntax:
                JKIL.types.isEmptyString ( pm_value[, ph_options] )
       Example 1: 
                var v = new Object();
                var b = JKIL.types.isEmptyString ( v ); // false
       Example 2: 
                var v = " ";
                var b = JKIL.types.isEmptyString ( v ); // false
       Example 3: 
                var v = "       ";
                var b = JKIL.types.isEmptyString ( v, {trimSequence: "\\s+"} ); // true
       Example 4: 
                var v = "xXxX";
                var b = JKIL.types.isEmptyString ( v, {trimSequence: "x+", ignore: ""} ); // false
       Example 5: 
                var v = "xXxX";
                var b = JKIL.types.isEmptyString ( v, {trimSequence: "x+", ignore: "i"} ); // true
       Example 6: 
                var v = new String("\n\t \v\n");
                var b = JKIL.types.isEmptyString ( v, {trimSequence: "\\s+"} ); // true
Parameters:
{mixed} pm_value
The value which is to be checked.

{hash} ph_options Optional
A hash of options:

{string} ph_options.trimSequence Optional, Default: ""
A regular expression representing the sequence that has to be removed from both sides of the string before checking its length (trimming is operated over multiple lines).

{string} ph_options.ignore Optional, Default: ""
Either "" (by default, trimming is case sensitive) or "i" (if case has to be ignored when trimming).
Returns:
{boolean} Returns true if the given value is an empty string literal or string object (after trimming or not, depending on options). Returns false otherwise (i.e. is not a string or is not an empty string).
Requires:
JKIL.types.getClassName()
See:
