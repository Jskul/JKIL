<static> {string} JKIL.types.getClassName(pm_value, ph_options)
Gets the value of the hidden [[Class]] property for the given value, i.e. the “class” name.
       Syntax:
                JKIL.types.getClassName ( pm_value[, ph_options] )
       Example 1: 
                var myObject = new MyObject();
                var className = JKIL.types.getClassName ( myObject, {checkConstructor: false} ); // "Object"
       Example 2: 
                var myObject = new MyObject();
                var className = JKIL.types.getClassName ( myObject, {checkConstructor: true} ); // "MyObject"
Parameters:
{mixed} pm_value
The value whose "class" is to be determined.

{hash} ph_options Optional
A hash of options:

{boolean} ph_options.checkConstructor Optional, Default: false
If true, for objects the method checks whether pm_value is an instance of its constructor property (i.e. if constructor has not been altered) and returns the name of this constructor.
Returns:
{string} A string describing the [[Class]] property (for instance "Object", "Array", "String", "Boolean" and so on).
Notice: window returns "Window" even for Google Chrome.
