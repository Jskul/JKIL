<static> {Array} JKIL.sortings.array.doShellSort(pa_array, ph_options)
Performs an in-place comparison sort of the given array using a Shell sort algorithm (worst performance equals O(n2), best performance equals O(n log2 n), average performance depends on gap sequence).
Notice: Uses a gap sequence of about one-third as recommended by Knuth, D.E. (1997) The Art of Computer Programming, Addison-Wesley.

Defined in: JKIL.sortings.array.doShellSort.js.

       Example 3:
                var a = JKIL.sortings.array.doShellSort ( numberArray, {ascending: true, comparator: JKIL.comparators.compareNumbers} );

       Example 1:

       Example 3:
                var b = JKIL.sortings.array.isSorted ( numberArray, {ascending: false} );
       Example 4:

Parameters:
{Array} pa_array
The array to be checked.

{hash} ph_options Optional
A hash of options:

{null|boolean} ph_options.ascending Optional, Default: null
If null, the method only checks whether the array is sorted. If true, the method checks if the array is sorted in an ascending manner. If false, it checks if the array is sorted in a descending manner.

{Function} ph_options.comparator Optional, Default: compareNumbers
A callback anonymous function or a reference to a function that manages the comparison between each pair of values. By default, comparison is made with the < and > operators.

TODO

{boolean} If true the array is sorted, if false the array is not sorted or is not sorted according to the given ph_options.ascending boolean value.
