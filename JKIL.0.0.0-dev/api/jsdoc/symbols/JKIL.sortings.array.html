<static> {Array} JKIL.sortings.array.bubbleSort(pa_array, ph_options)
Performs an in-place comparison sort of the given array using a bubble sort algorithm (worst and average performances equal O(n2), best performance equals O(n)).

Defined in: JKIL.sortings.array.bubbleSort.js. 
	Syntax:
				JKIL.sortings.array.bubbleSort ( pa_array, ph_options )
	Example 1:
             var a = JKIL.sortings.array.bubbleSort ( numberArray );
    Example 2:
             var a = JKIL.sortings.array.bubbleSort ( numberArray, {ascending: false} );
    Example 3:
             var a = JKIL.sortings.array.bubbleSort ( numberArray, {ascending: true, comparator: JKIL.comparators.compareNumbers} );
Parameters:
{Array} pa_array
The array to be sorted.

{hash} ph_options Optional
A hash of options:

{boolean} ph_options.ascending Optional, Default: true
If true, the array is sorted in an ascending manner. If false, the array is sorted in a descending manner.

{Function} ph_options.comparator Optional, Default: compareNumbers
A callback anonymous function or a reference to a function that manages the comparison between each pair of values. By default, comparison is made with the < and > operators.
