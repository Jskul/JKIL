Compares two objects according to a property when it is present in both of them (as an own property).

Defined in: JKIL.comparators.compareObjectsByKey.js. 
       Syntax:
                JKIL.comparators.compareObjectsByKey ( pO_a, pO_b, ps_key, pf_comparator )
